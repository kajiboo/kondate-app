# 晩ごはん献立プランナー

家族向けの晩ごはん献立Webアプリです。

## できること

- 期間を指定して献立を生成
- 大人と子供の人数でカロリー・費用・食材量を目安計算
- 日ごとにジャンル、手間、弁当向き、希望献立を指定
- 特定の日の献立をリストから変更、または別案に差し替え
- 主食材・野菜・味付け・調理法を組み合わせた幅広い献立提案
- `recipe_dish_names_2000.csv` の料理名を献立候補として読み込み
- 各献立から詳しいレシピ検索を開く
- アレルギー、子供NG食材、使いたくない食材を除外
- 魚多め、肉少なめ、時短多めなどの固定ルール
- 買い物リストとレシピ表示
- 過去の献立ログを保存・復元
- 合言葉による簡易ロック
- ホーム画面追加用のPWA設定

## 使い方

`index.html` をブラウザで開きます。
初回は合言葉を設定してください。

GitHub Pagesへアップロードする場合は、`recipe_dish_names_2000.csv` も `index.html` と同じ場所に置いてください。

## iPhoneのホーム画面に追加する場合

iPhoneでホーム画面に追加するには、Safariで開けるURLとして配信する必要があります。
HTTPSの静的ホスティングに置くと、ホーム画面追加とオフラインキャッシュが使いやすくなります。

## 注意

合言葉は家族利用向けの簡易ロックです。
本格的なユーザー認証や家族間共有が必要な場合は、サーバーやクラウド保存を追加してください。

レシピ検索は料理名で外部検索を開く方式です。
GitHub Pagesだけで動かすため、検索APIキーをアプリ内に埋め込む作りにはしていません。
